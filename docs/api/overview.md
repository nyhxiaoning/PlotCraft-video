# API 概述

PanelFlow 核心服务 API。

## 核心服务

| 服务                     | 描述             | 状态    |
| ------------------------ | ---------------- | ------- |
| `aiService`              | 统一 AI 文本生成 | ✅ 稳定 |
| `imageGenerationService` | 多提供商图像生成 | ✅ 稳定 |
| `ttsService`             | 文本转语音合成   | ✅ 稳定 |
| `pipelineService`        | 端到端流水线     | ✅ 稳定 |
| `storyboardService`      | 分镜管理         | ✅ 稳定 |
| `characterService`       | 角色管理         | ✅ 稳定 |
| `subtitleService`        | 字幕生成与导出   | ✅ 稳定 |

## 服务架构

```
┌─────────────────────────────────────────┐
│             核心服务层                   │
├─────────────────────────────────────────┤
│  ┌──────────────┐  ┌──────────────┐    │
│  │  AI 文字生成  │  │  图像生成    │    │
│  │  (多模型)    │  │  (多模型)    │    │
│  └──────┬───────┘  └──────┬───────┘    │
│         │                  │             │
│  ┌──────▼───────┐  ┌──────▼───────┐    │
│  │  TTS 语音    │  │  唇形同步    │    │
│  └──────────────┘  └──────────────┘    │
│  ┌──────────────┐  ┌──────────────┐    │
│  │  流水线引擎   │  │  视频导出    │    │
│  └──────────────┘  └──────────────┘    │
└─────────────────────────────────────────┘
```

## 类型安全

所有服务均使用 TypeScript 接口，提供完整类型支持。
