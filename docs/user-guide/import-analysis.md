# 导入与分析

上传并解析源内容。

## 支持的格式

| 格式   | 扩展名        | 说明               |
| ------ | ------------- | ------------------ |
| 小说   | `.txt`、`.md` | UTF-8/GBK 自动检测 |
| 脚本   | `.json`       | 结构化 JSON        |
| 提示词 | 纯文本        | 自定义 AI 提示词   |

### 小说格式示例

```markdown
# 第一章：相遇

阳光明媚的早晨，小明走在校园里。

小明：今天天气真好！

突然，一个女孩从转角出现。
```

### JSON 脚本格式

```json
{
  "title": "脚本标题",
  "scenes": [
    {
      "id": "scene_1",
      "description": "场景描述",
      "dialogue": [{ "character": "小明", "text": "你好！", "emotion": "happy" }]
    }
  ]
}
```

## 导入流程

1. 点击工作流面板上的 **导入**
2. 选择输入类型：小说 / 脚本 / 提示词
3. 上传文件或输入文本
4. 点击 **分析**

## 分析功能

| 功能     | 说明             |
| -------- | ---------------- |
| 章节检测 | 识别章节边界     |
| 场景分割 | 将内容分割为场景 |
| 角色提取 | 识别并命名角色   |
| 情感检测 | 分析情感基调     |
| 对话提取 | 提取角色对话     |

## 最佳实践

- 使用 UTF-8 编码避免乱码
- 用 `# 章节` 标题或空行分隔章节
- 对话前包含角色名称（如 `小明：今天天气真好！`）
