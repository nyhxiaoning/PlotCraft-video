# 工作流概览

PanelFlow 七步工作流将你的故事转化为视频内容。

## 工作流程图

```
┌──────────────────────────────────────────────────────────────────────┐
│                         PanelFlow 七步工作流                          │
├──────────────────────────────────────────────────────────────────────┤
│                                                                       │
│   📥 导入    →   🧠 AI分析    →   📝 脚本生成   →   🎬 分镜设计       │
│      │              │               │                │              │
│      ↓              ↓               ↓                ↓              │
│  小说/脚本/      多模型并行     结构化视频      自动分镜图            │
│  提示词上传     内容分析       脚本输出       含镜头/构图            │
│                                                                       │
│   🎭 角色设计  →   🖼️ 批量渲染   →   📤 导出                       │
│        │               │               │                            │
│        ↓               ↓               ↓                            │
│   AI角色参考图    多模型并行     MP4/WebM/MOV                         │
│   种子一致性      场景渲染       字幕+唇形同步                         │
│                                                                       │
└──────────────────────────────────────────────────────────────────────┘
```

## 步骤详解

### 第一步：导入 📥

上传源内容：小说 TXT/MD、JSON 脚本或自定义提示词。AI 自动检测编码（UTF-8/GBK）和章节边界。

### 第二步：AI 分析 🧠

多模型并行理解内容，识别章节结构、角色关系、场景类型、情感基调。

### 第三步：脚本生成 📝

基于分析生成结构化视频脚本，包含分镜头描述、对话台词、场景描述。

### 第四步：分镜设计 🎬

将脚本转换为可视化分镜，AI 提供镜头角度建议和构图参考。

### 第五步：角色设计 🎭

AI 创建角色参考图，通过种子机制确保多场景一致性。

### 第六步：批量渲染 🖼️

多模型并行渲染场景图、角色肖像、背景素材。

### 第七步：导出 📤

合成时间轴、嵌入字幕、TTS 唇形同步，导出 MP4/WebM/MOV。
